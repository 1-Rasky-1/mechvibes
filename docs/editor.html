For quicker and more easier to understand those steps below, download demo pack and check how it looks like.

1:

Use free tool like Audacity to determine start and length of a sound in audio file

2:

Open Mechvibes Editor, select Key define mode: "Single file"

3:

Select a key that you want to assign sound

4:

Type start and length as you selected in Audacity

5:

Repeat those step until you fill out all the keys in editor

6:

Click Export to get the config.json file contains your key definitions

7:

Create a folder and copy the exported config.json file and sound file into it

8:

Click Browse custom packs

9:

Copy your custom pack folder into custom folder (create custom folder if it not exists)

10:

Restart Mechvibes app and your custom pack will appears in sound pack list

For quicker and more easier to understand those steps below, download demo pack and check how it looks like.

1:

Create an empty folder, copy all your audio files into it

2:

Open editor

3:

Select a key and type in the audio file's name that you want to assign to selected key

4:

Repeat until you fill out all the keys

5:

Click Export to get the config.json file that contains your key definitions

6:

Place config.json inside your custom pack folder

7:

Copy you custom pack folder into app custom folder

8:

Restart Mechvibes app and your custom pack will appears in sound pack list
